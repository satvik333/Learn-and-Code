Integrate Geocoding API in your application:  

Get API reference from: https://developers.google.com/maps/documentation/javascript/geocoding

Create a simple console application where 
User will give place as an input and you need to return latitude and longitude of that place and show it in application. 

Example: 
UserInput: SilkBoard 

Output: 
Latitude: 12.9177
Longitude:  77.6238
